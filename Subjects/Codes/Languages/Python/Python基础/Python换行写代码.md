Python中一般是一行写完所有代码，如果遇到一行写不完需要换行的情况，有两种方法：

1.在该行代码末尾加上续行符“ \”（即空格+\）；

test = 'item_one' \

'item_two' \

  'tem_three'

输出结果：'item_oneitem_twotem_three'

2.加上括号，() {}  []中不需要特别加换行符：

test2 = ('csdn '

'cssdn')

输出结果：csdn cssdn
