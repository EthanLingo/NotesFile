# 常用的正则表达式





中文字符

```
[\\\\u4e00-\\\\u9fa5]
```



双字节字符

```
[^\\\\x00-\\\\xff]
```



邮箱

```
\w[-\w.+]*@([A-Za-z0-9][-A-Za-z0-9]+\.)+[A-Za-z]{2,14}
```



网址URL

```
^((https|http|ftp|rtsp|mms)?:\/\/)[^\s]+
```



手机号

```
0?(13|14|15|18)[0-9]{9}
```



浮点数

```
-?([1-9]\\\\d*.\\\\d*|0.\\\\d*[1-9]\\\\d*)
```



整数

```
-?[1-9]\\\\d*
```



QQ

```
[1-9]([0-9]{5,11})
```



IP

```
(25[0-5]|2[0-4]\\\\d|[0-1]\\\\d{2}|[1-9]?\\\\d)\.(25[0-5]|2[0-4]\\\\d|[0-1]\\\\d{2}|[1-9]?\\\\d)\.(25[0-5]|2[0-4]\\\\d|[0-1]\\\\d{2}|[1-9]?\\\\d)\.(25[0-5]|2[0-4]\\\\d|[0-1]\\\\d{2}|[1-9]?\\\\d)
```



身份证

```
\\\\d{17}[\\\\d|x]|\\\\d{15}
```



日期格式

```
\\\\d{4}(\-|\/|.)\\\\d{1,2}\1\\\\d{1,2}
```



中文、数字或字母

```
[A-Za-z0-9_\-\\\\u4e00-\\\\u9fa5]+
```
